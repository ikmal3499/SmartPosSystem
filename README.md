# 🛒 Smart POS System with AI Recommendations

A modern Point of Sale (POS) system built with React frontend and Spring Boot backend, featuring AI-powered product recommendations that suggest complementary items to customers in real-time.

## 🌟 Features

- **Product Management**: Add products to cart with one click
- **Real-time Cart**: View cart items and running total
- **AI Recommendations**: Smart system suggests complementary products based on cart contents
  - Example: "Customers who bought Coffee also bought Cake"
- **Checkout System**: Process transactions and generate receipts
- **Responsive Design**: Modern UI with hover effects and animations
- **Transaction History**: Generate unique transaction IDs for each sale

## 🛠️ Tech Stack

### Frontend
- **React** (with Vite) - Modern JavaScript library
- **Axios** - HTTP client for API calls
- **Inline CSS** - Styling with custom gradients and animations

### Backend
- **Java 17+** - Programming language
- **Spring Boot 3.x** - Backend framework
- **Spring Web** - REST API framework
- **Maven** - Dependency management

## 📋 Prerequisites

Before running this project, ensure you have:

- **Java 17 or higher** installed ([Download JDK](https://adoptium.net/))
- **Node.js 18+** and npm installed ([Download Node.js](https://nodejs.org/))
- **VS Code** or any preferred IDE
- **Spring Boot Extension Pack** (for VS Code)
